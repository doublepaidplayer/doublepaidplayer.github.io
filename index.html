第2章 “欺骗”大脑的方法

2-1 无论是谁都会忘记

不论聪明与否，先让一组人记住一些信息，然后检测对这些信息记忆的维持时间。实验结果表明：短期记忆的信息忘记速度总是大致相同的，不会因为人不同而出现明显差异

记住的信息总是在固定某一时间段被我们遗忘，我们想提早忘记某些信息也是做不到的

实验的遗忘曲线表明：

  • 人脑不是匀速忘记信息的
  • 最容易忘记的时间是刚刚记住的时候——记住信息的4小时内
  • 这4小时内我们会一口气忘记一半的内容
  • 剩余的记忆会维持较长的时间

故，考试临时抱佛脚时，前一天晚上彻夜苦读是比不上考试当天早起磨枪的。不是考试前4小时以内记住的东西会在考试开始时忘掉一大半

补充：组块化

背电话号码时分成几块“连续的数字”进行记忆，比如

18812662456 => 1881-2662-456

背单词时将词组分类整理后再背，比如

get at / get out / get over / get up 分成一组

学习中对知识信息进行分类整理是非常重要的步骤，数学计算过程中常出现马虎计算错误的人，其笔算过程常常也是杂乱无章的

2-2 好方法？坏方法？

上面提到的“忘记的速度不会因人而异”是在有相同外部条件下才会满足的，实际上每个学生的学习成绩不同，是因为他们有不同的“外部情况”

什么情况会加速信息的遗忘程度呢？

最容易导致原有的记忆提前消失的活动，是添加新的记忆，即将知识一股脑地硬塞进大脑中。如果往脑中塞入过量信息，记忆的效果就会变差，因为人脑一次能记住的信息是有限的

新添加的记忆会加速短期旧记忆的遗忘，而短期前的旧记忆又会影响新添加记忆的牢固程度，新旧记忆互相影响的现象即“记忆的干扰”

存在于人脑中的一个个记忆片段并非毫无关联、独立存在，有时它们互相抑制，有时候互相合作得到巩固。故，错误的记忆方法，毫无准备地将大量知识塞入脑中就会使记忆混乱、模糊不清，造成失误

举个例子：

突击考试，100道难度较大的题（100道题的考试范围是大纲中分散的），给一天的时间复习。花大量时间去试图把100道题全背下来，与踏踏实实地只细心准备30道题相比，通常后者能得到更高的分数

不仅是考试，在平时学习过程中也要尽量避免在一天内向脑输入大量新知识。而学习的重点，实际应该放在“复习”上。在力所能及的范围内没有压力地记住自己所能记住的内容，才是符合记忆性质的学习方法

补充：经验谈2

  • 买参考书时先提前翻一翻，要有好的第一印象，以便维持自己的学习热情
  • 勿好高骛远，不要把学习目标定的太高。就日常学习而言，达成目标所带来的成就感会适当地刺激人脑中的A10神经产生快乐情绪。合理设定目标，在长时间内一次次获得成就感，进而提高学习热情
  • 学习完成后进行打卡，明确地看到自己应该做的事情做好了，对维持学习热情也有帮助
  • “复习”非常重要，不能以“还要学习其他知识”为由降低复习优先级。本书作者认为：预习、学习、复习 的比例在 1/4 : 1 : 4 比较妥当

2-3 反复记忆的效果

无论是谁都能通过反复记忆，让自己“看起来好像是记忆力提高了”。那些我们觉得自己已经完全忘记的信息，其实都完整地保存在“无意识的世界里”，只不过它们属于“潜在的痕迹”，所以才无法被想起来

当我们反复记忆时，这些潜在痕迹会悄无声息地帮助我们记住知识，从而提高考试成绩，复习可以降低我们忘记知识的速度

补充：经验谈3

“我到底为了什么而学习”

自己的人生价值和意义是什么？对现状和未来感到绝望吗？找不到自己的方向吗？人生的目标是什么？

在迷惘、萎靡与消沉、不清楚自己真正想要什么的时间段，正是学习能力上和他人产生差距的关键时期

2-4 蛮干终究是徒劳

复习需要注意三点，第一点是复习的时机

一、复习的时机，什么时候复习比较好

潜在记忆的保存时间只有一个月左右，不在一个月内复习学到的知识的话潜在记忆就无法发挥作用。所以并非任何时候复习都有效果，最迟也要在一个月内就开始复习

为什么无意识的记忆会有“保质期”呢？关键还在海马体

把海马体看作是信息审查工厂，信息种类决定该信息能在这家工厂停留的时间，短的大概只能停留一个月左右。海马体会在这一个月内分类整理这些信息，判定哪些是进入大脑皮质的必要信息

间隔了一个月甚至更久时间才去复习的知识，对海马体而言和新学习的知识没有什么不同

另外，在首次复习中输送进海马体的信息越多，让海马体归类为必要信息的可能性就越大。在复习时也要像初次学习那样用功，不仅要用眼睛看，还要动笔写、出声读，尽可能调动自己的感官，视觉、听觉、触觉传递的信息都会对海马体起到刺激的作用

插一条笔记

花时间记笔记确实是对学习有用的，但是有时候会太过关注“记笔记”这件事本身，寻枝摘叶，把注意力集中在自己做的笔记有没有不整洁、写字有没有一行对齐、字写
的好不好看等等细枝末节的事（所谓的强迫症发作了）

尤其在网上看到过很多次有些女生晒出自己平时写的五颜六色字迹的笔记多么美观，多么详细，最后哭丧着自己考砸了的样子，其实就是将精力集中错了位置。这种行为只是在欣赏写的笔记的“外壳”，并没有深入知识的“内核”，甚至就只是照抄书本上的文字，完全没有吸收知识的作用。更严重的是，很多时候这些笔记写了就放在那了，不会二次翻阅，那这种行为更属于纯粹的浪费时间，所谓“感动当时作笔记的自己”

回想起来，我以前练习硬笔书法的时候，也有这种情况。当时学习的时候作笔记，会特别关注自己每一个字写的怎么样，一旦出现错别字，或者某个笔画偏了角度、字的结构不好、写长串句子字的布局乱糟糟，自己看着就会特别闹心。所以当时特别追求写“好看的笔记”，写一篇废掉大量的时间。当后来我写字的水平逐渐提高，再回过来看“当时觉得好看的笔记”，又会觉得字的水平太低，不忍直视，甚至想把笔记本扔垃圾桶里，那些“精雕细琢”的笔记也没有发挥它们应有的作用

不可否认学习时作笔记确实比纯看资料更有利于记忆。但是就作笔记的方式而言，传统纸笔的速度明显是不如键盘打字的，而且电子设备显示的打印体也不会再让人分散精力到字迹是否工整的问题上，电脑上的笔记想要增删改查、添加标注也很方便，用滚动条的浏览体验也比翻页要好，不想要的笔记可以直接删除无需用碎纸机。另外直接复制到手机上查看的话也很方便，不用再去费精力翻堆积如山的笔记本。而要定位具体的哪一篇笔记、哪一段话、哪一个关键点，电子设备都是比纸质笔记方便的，可以说传统纸质笔记已经没有什么优点了

现在我的学习方法：

一、第一次学习时，将那些首次接触的、还未能理解的概念性的关键词在草稿纸上写一写，基本上随意写写，不会在意工整程度、排版什么的。往下若遇到与之前写的关键词有联系的新的关键词，会给它们加个箭头辅助记忆。纸笔的工作除了基础的给数学计算打草稿，仅仅起着加强初次记忆的作用，基本不会再看第二次（除非记下了特别重要的参考资料没有提到的内容）

二、第一次复习时，看和初次学习时一样的参考资料。看的同时将那些重要的以及描述比较晦涩难懂的内容进行分解精简，然后贴在 markdown 里，一定避免一个字不改地复制或照着原文打，尽量用自己的话复述一遍打出来。将自己写的 markdown 文件整理好，以后复习时就不用再去费力翻原来的资料了，更不用从头到尾通读原文

三、后续的复习基本上只看自己写的 markdown 文档，边看边思考

结合海马体的性质，本书作者推荐的复习计划：

第几次复习 在第几天（假设第一次学习在 Day1）
1 Day2
2 Day9
3 Day23
4 Day53

复习间隔时间分别为 1天 / 7天 / 14天 / 30天，一个月为完整的学习周期，学习1次后复习4次。这样做已经达到高效复习的目的，没有必要再复习更多次。复习计划安排得再满，对最终学习效果也不会产生任何影响，仅仅是浪费精力

二、复习的内容

复习需要注意的第二点，是一定要复习同样的内容。这个范围是非常小的，也就是说不仅仅是信息的相关性，而且连参考资料都要一致。即便科目相同，换了参考书，复习效果也会不佳

选定了参考的资料，就要一心一意地将它看完。哪怕后来有身边的人推荐、或自己发现了比它更好的，也尽量不要朝秦暮楚，频繁更换参考资料会影响学习效果

2-5 人脑更重视输出

复习需要注意的第三点是

三、人脑更重视输出而不是输入

实验：让一组参与者记忆非常难见到的外语单词

具体的实验过程：让所有参与者学习所有单词，然后马上进行测试（当然对于这些从没见过的单词，第一次没人能得满分）。接着将参与者分为4组，用不同的策略继续进行记忆与测试，分组如下：

  1. 如果没能拿到满分，就把所有单词重新背一遍，再重新接受之前同样的测试。如果还是没拿满分，就再去背所有单词，再来接受测试。重复“学习并测试”的过程，直到拿满分为止
  2. 第2组轻松一些，重新记忆所有单词太麻烦了，所以他们没拿到满分的只需要背诵上次测试中出错的单词，然后再接受全部单词的测试。如果还不能拿满分，就继续背诵出错的单词，并重新接受测试。如此反复，直到拿满分
  3. 第3组和第2组恰恰相反，重新记忆单词时背诵全部的单词，但测试只会测试上一次测试中出错的部分。如果不能拿满分，就重新记忆所有的单词，然后再次测试出错的部分。如此反复，直到拿满分
  4. 第4组采用日常生活中最普遍的做法，即只重新记忆上次测试中出错的单词，测试时也只考出错的部分，一直重复这个过程，直至不再出错
重新背的单词 测试的单词
第1组 全部单词 全部单词
第2组 出错的单词 全部单词
第3组 全部单词 出错的单词
第4组 出错的单词 出错的单词

实验结果：

就记忆速度而言，4组基本不存在差异。每组重复记忆单词的次数都是一样的

但在一段时间后，再次测试参与者们对单词的掌握程度，成绩就出现了十分明显的差距：第1组和第2组成绩在80分左右，而第3组和第4组只有大概40分左右

成绩较好的第1组和第2组在重新测试时考查了全部的单词，而第3组虽然重新背了全部的单词，但只测试了之前出错的部分

人脑中存在“输入”和“输出”两种操作，背单词的行为相当于输入，提取保存在大脑中的单词去解答试题的行为相当于输出操作。没有输入，就没有输出，但从该实验结果可以看出，脑更重视“输出”一方。人脑的模式是“输出依赖性”

再谈海马体的角度，做出的判断其实是“这个信息竟然会被如此频繁调用，看来必须要记住它”
